# Neth-Sawan 🤟 👁️

**Neth-Sawan** is an AI-powered accessibility platform specifically designed to empower individuals with hearing impairments. By leveraging modern web technologies and Artificial Intelligence, it bridges the communication gap between the hearing-impaired community and the world.

---

## 🌟 Key Features

* **Real-time Speech-to-Sign Language:** Converts spoken language into visual sign language animations or images instantly.
* **AI Object Recognition:** Uses the camera to identify objects and surroundings, providing visual feedback to the user.
* **Sound Visualization:** Transforms ambient sounds into visual alerts or waveform animations (e.g., detecting a doorbell or emergency sirens).
* **Emergency Contact System:** A quick-access module to notify pre-set emergency contacts with location data.
* **AI Chat Assistant:** An integrated AI bot to assist users in navigating daily tasks through text and visual cues.

## 🛠️ Tech Stack

* **Frontend:** React.js, Tailwind CSS
* **State Management:** React Hooks (useState, useEffect)
* **AI Integration:** Google Generative AI SDK (Gemini API)
* **Icons & UI:** Lucide-React, Framer Motion
* **Deployment:** Vercel / GitHub Pages
